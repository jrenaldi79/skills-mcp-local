# PDF Helper Skill

A skill for working with PDF documents.

## Capabilities

- Extract text from PDF files
- Get PDF metadata (page count, author, etc.)
- Convert PDF pages to images

## Requirements

- `poppler-utils` package installed
- Python 3.8+

## Usage

1. Place PDF files in a known location
2. Ask Claude to analyze or extract content from the PDF
3. Follow the prompts for specific operations
